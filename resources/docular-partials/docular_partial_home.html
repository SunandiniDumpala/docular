Docular : NodeJS Documentation Generation

Docular is a * BETA * NodeJS package that performs documentation generation based on AngularJS's documentation generation implementation.

Docular will parse, manipulate, and render documentation embedded within your script files or within standlone non-script files. The documentation can be viewed within a webapp through NodeJS or can be moved to the root of a web server.

Docular is best used through the Grunt plugin "grunt-docular".
